## 🎯 الهدف

إضافة نظام شامل لإدارة **سجلات الصيانة** بحيث:

* يتم إنشاء سجل لكل عملية صيانة (وقائية، تصحيحية، معايرة، طارئة)
* يحتوي السجل على تفاصيل كاملة (من نفّذها، الوقت، الأدوات، التكلفة، الملاحظات)
* يرتبط مباشرة بكل جهاز Equipment
* يُحدث تلقائيًا حالة الجهاز (Active / Under Maintenance / Completed)
* يظهر في **Dashboard** و **Equipment Details Drawer**

## ⚙️ 1. قاعدة البيانات (Database Schema)

### 🧱 جدول الصيانة الجديد

```sql
CREATE TABLE maintenance_records (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  equipment_id UUID REFERENCES equipment(id) ON DELETE CASCADE,
  technician_id UUID REFERENCES users(id),
  maintenance_type TEXT CHECK (maintenance_type IN ('Preventive','Corrective','Calibration','Emergency')) DEFAULT 'Preventive',
  start_date TIMESTAMP DEFAULT NOW(),
  end_date TIMESTAMP,
  description TEXT,
  actions_taken TEXT,
  parts_used TEXT,
  cost NUMERIC(10,2) DEFAULT 0,
  status TEXT CHECK (status IN ('In Progress','Completed','Pending Verification')) DEFAULT 'In Progress',
  verification_status TEXT CHECK (verification_status IN ('Verified','Rejected','Pending')) DEFAULT 'Pending',
  verified_by UUID REFERENCES users(id),
  verified_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

✅ يدعم أنواع الصيانة المختلفة
✅ يربط الصيانة بالمعدات والفنيين
✅ فيه متابعة التحقق (Verification)

## 🔄 2. تحديث جدول المعدات (Equipment Sync)

لكي يتفاعل النظام تلقائيًا:

```sql
CREATE OR REPLACE FUNCTION update_equipment_status_after_maintenance()
RETURNS TRIGGER AS $$
BEGIN
  IF NEW.status = 'Completed' THEN
    UPDATE equipment SET status = 'Active', last_maintenance_date = NOW(), next_due_date = NOW() + INTERVAL '6 months'
    WHERE id = NEW.equipment_id;
  END IF;
  RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER trg_equipment_status_update
AFTER UPDATE ON maintenance_records
FOR EACH ROW
EXECUTE FUNCTION update_equipment_status_after_maintenance();
```

💡 كل ما يتم إنهاء عملية صيانة → يحدث الجهاز تلقائيًا
(Active + تحديث التاريخ + حساب موعد الصيانة القادم)

## 🧠 3. الـ API Endpoints

### 🔹 إنشاء سجل صيانة

`POST /maintenance-records`

```js
app.post('/maintenance-records', async (req, res) => {
  const { equipment_id, technician_id, maintenance_type, description, actions_taken, parts_used, cost } = req.body;

  const result = await db.query(`
    INSERT INTO maintenance_records (equipment_id, technician_id, maintenance_type, description, actions_taken, parts_used, cost)
    VALUES ($1,$2,$3,$4,$5,$6,$7) RETURNING *;
  `, [equipment_id, technician_id, maintenance_type, description, actions_taken, parts_used, cost]);

  await db.query(`UPDATE equipment SET status='Under Maintenance' WHERE id=$1`, [equipment_id]);
  res.json(result.rows[0]);
});
```

---

### 🔹 إنهاء عملية الصيانة

`PATCH /maintenance-records/:id/complete`

```js
app.patch('/maintenance-records/:id/complete', async (req, res) => {
  const { id } = req.params;
  const { verified_by } = req.body;

  const result = await db.query(`
    UPDATE maintenance_records
    SET status='Completed', verification_status='Verified', verified_by=$1, verified_at=NOW(), end_date=NOW()
    WHERE id=$2 RETURNING *;
  `, [verified_by, id]);

  await db.query(`
    UPDATE equipment
    SET status='Active', last_maintenance_date=NOW(), next_due_date=NOW() + INTERVAL '6 months'
    WHERE id=(SELECT equipment_id FROM maintenance_records WHERE id=$1)
  `, [id]);

  res.json(result.rows[0]);
});
```

---

### 🔹 عرض كل السجلات

`GET /maintenance-records`

```js
app.get('/maintenance-records', async (req, res) => {
  const { equipment_id, technician_id, status } = req.query;
  let query = `SELECT m.*, e.name AS equipment_name, u.name AS technician_name
               FROM maintenance_records m
               LEFT JOIN equipment e ON m.equipment_id = e.id
               LEFT JOIN users u ON m.technician_id = u.id
               WHERE 1=1`;
  const params = [];

  if (equipment_id) {
    params.push(equipment_id);
    query += ` AND m.equipment_id = $${params.length}`;
  }
  if (technician_id) {
    params.push(technician_id);
    query += ` AND m.technician_id = $${params.length}`;
  }
  if (status) {
    params.push(status);
    query += ` AND m.status = $${params.length}`;
  }

  query += ' ORDER BY m.start_date DESC';

  const result = await db.query(query, params);
  res.json(result.rows);
});
```
