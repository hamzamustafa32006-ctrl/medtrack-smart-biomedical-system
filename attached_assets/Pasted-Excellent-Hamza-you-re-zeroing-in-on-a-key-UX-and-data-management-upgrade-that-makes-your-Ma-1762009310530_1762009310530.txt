Users should be able to:

## ⚙️ 1. Backend Implementation (Node.js + PostgreSQL)

### ✅ Endpoint: `GET /equipment`

Example:

```
/equipment?status=Under%20Maintenance&search=ventilator
```

### 📜 Code (Express)

```js
app.get('/equipment', async (req, res) => {
  const { status, search } = req.query;

  let query = `
    SELECT e.id, e.name, e.model, e.serial_number, e.status, e.next_due_date,
           f.name AS facility_name
    FROM equipment e
    LEFT JOIN facilities f ON e.facility_id = f.id
    WHERE 1=1
  `;
  const params = [];

  // Filter by maintenance status
  if (status) {
    params.push(status);
    query += ` AND e.status = $${params.length}`;
  }

  // Text-based search
  if (search) {
    params.push(`%${search.toLowerCase()}%`);
    query += ` AND (
      LOWER(e.name) LIKE $${params.length}
      OR LOWER(e.model) LIKE $${params.length}
      OR LOWER(e.serial_number) LIKE $${params.length}
      OR LOWER(f.name) LIKE $${params.length}
    )`;
  }

  query += ' ORDER BY e.name ASC';

  try {
    const { rows } = await db.query(query, params);
    res.json(rows);
  } catch (err) {
    console.error('Error filtering equipment:', err);
    res.status(500).json({ error: 'Failed to filter equipment.' });
  }
});
```

✅ Safe — uses parameterized SQL
✅ Flexible — either `status`, `search`, or both
✅ Clean sorting by name

### 🧱 Recommended Indexes (PostgreSQL)

Improve query performance:

```sql
CREATE INDEX IF NOT EXISTS idx_equipment_status ON equipment(status);
CREATE INDEX IF NOT EXISTS idx_equipment_search 
ON equipment USING gin(to_tsvector('english', name || ' ' || model || ' ' || serial_number));
```

```tsx
  <input
    type="text"
    placeholder="Search equipment..."
    value={search}
    onChange={(e) => setSearch(e.target.value)}
    className="border p-2 rounded-md w-64"
  />

  <button
    className="bg-blue-600 text-white px-4 py-2 rounded-md"
    onClick={() => fetchEquipment(status, search)}
  >
    Apply
  </button>
</div>
```

### 🔹 Fetch Function

```js
const fetchEquipment = async (status, search) => {
  const params = new URLSearchParams();
  if (status) params.append('status', status);
  if (search) params.append('search', search);

  const res = await fetch(`/equipment?${params.toString()}`);
  const data = await res.json();
  setEquipment(data);
};
```
