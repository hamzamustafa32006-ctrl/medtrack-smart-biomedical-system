Feature Goal
Give users (admins and supervisors) the ability to:
Create and manage facilities (e.g., hospitals, labs, departments).
Add locations (e.g., Radiology Room 1, ICU Room 3) inside each facility.
Assign equipment to those locations.
Filter all maintenance tasks and alerts by facility or location.
Hierarchy
Facility → Locations → Equipment → Maintenance Tasks
⚙️ 2. Database Layer
Tables
CREATE TABLE facilities (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  code TEXT UNIQUE,
  address TEXT,
  contact_person TEXT,
  phone TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE locations (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  facility_id UUID REFERENCES facilities(id) ON DELETE CASCADE,
  name TEXT NOT NULL,
  floor TEXT,
  room TEXT,
  department TEXT,
  notes TEXT,
  UNIQUE (facility_id, name)
);

ALTER TABLE equipment
  ADD COLUMN facility_id UUID REFERENCES facilities(id),
  ADD COLUMN location_id UUID REFERENCES locations(id);
Key Relationships
One facility → many locations
One location → many equipment items
Deleting a facility cascades to its locations
🧠 3. Backend API Design
Facility Endpoints
Method	Endpoint	Description
GET	/facilities	List all facilities
POST	/facilities	Create facility
PATCH	/facilities/:id	Update facility info
DELETE	/facilities/:id	Delete facility
GET	/facilities/:id/locations	Get all locations under a facility
Location Endpoints
Method	Endpoint	Description
POST	/facilities/:facility_id/locations	Create location
PATCH	/locations/:id	Update location
DELETE	/locations/:id	Delete location

🧱 4. Frontend Implementation
a. Facility Management Page
List all facilities with summary stats (e.g., equipment count, open tasks).
“Add Facility” button → modal form.
Edit/Delete actions inline.
b. Location Management (nested view)
When viewing a facility, show a Locations tab.
Table listing all locations.
“Add Location” button → modal with name, floor, room, department.
Inline edit/delete buttons.
c. Equipment Form Integration
Add two dropdowns in the Equipment creation form:
<Label>Facility</Label>
<Select onChange={e => setFacility(e.target.value)}>
  {facilities.map(f => <option key={f.id} value={f.id}>{f.name}</option>)}
</Select>

<Label>Location</Label>
<Select disabled={!facilityId} onChange={e => setLocation(e.target.value)}>
  {locations.filter(l => l.facility_id === facilityId)
    .map(l => <option key={l.id} value={l.id}>{l.name}</option>)}
</Select>
